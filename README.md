# Graficacion_Pygame_2_AxelGarcia
Proyectos Pygame 2

Sesion4_Ej1:Simulador de un círculo que se mueve horizontalmente rebotando en los bordes de la pantalla. Cada rebote aumenta su velocidad en un 10%, haciendo que se mueva más rápido con el tiempo. Desarrollado con Pygame.

Sesion4_Ej2:Programa de animación que muestra un círculo azul en pulsación constante. El círculo se expande y contrae de forma cíclica entre un radio mínimo y máximo definido, creando un efecto visual rítmico de latido. Desarrollado con la biblioteca Pygame para la representación gráfica en tiempo real.

Sesion4_Ej3:Simulación física que representa un objeto bajo los efectos de la gravedad. Un círculo verde cae desde la parte superior de la pantalla y rebota en una superficie horizontal, perdiendo progresivamente energía en cada colisión hasta alcanzar el reposo. El modelo incorpora principios de aceleración gravitacional y coeficiente de restitución para simular el comportamiento real del movimiento.

Sesion5_Ej1:Este programa permite visualizar una imagen con capacidad de ajuste interactivo de su tamaño. Los usuarios pueden aumentar o disminuir la escala de la imagen utilizando las teclas '+' y '-', manteniendo siempre las proporciones originales. La aplicación incluye una superposición informativa que muestra el factor de escala actual y proporciona instrucciones de uso.

Sesion5_Ej2:Programa que muestra una animación cíclica usando una hoja de sprites. Un personaje se anima automáticamente pasando por diferentes frames para simular movimiento. Incluye información sobre el frame actual y el total de frames.

Sesion5_Mini:Aplicación interactiva que modela el comportamiento de una nave espacial en un entorno bidimensional. El sistema implementa control de orientación mediante seguimiento del cursor y propulsión direccional con gestión de inercia y fricción atmosférica.

Sesion6_Ej1:Este programa implementa un mecanismo de detección de colisiones entre formas geométricas que proporciona retroalimentación visual inmediata mediante cambio de color. Un rectángulo controlado por el usuario mediante teclas de dirección interactúa con un círculo estático, cambiando su color al hacer contacto.

Sesion6_Ej2:Sistema interactivo que implementa un mecanismo de recolección de elementos en un espacio bidimensional. El usuario controla una entidad rectangular que interactúa con objetos circulares, incrementando un contador por cada elemento recolectado y activando la generación procedural de nuevos objetivos.

Sesion6_Ej3:Sistema interactivo que desafía al usuario a navegar un agente azul a través de un campo de obstáculos rojos en movimiento. Los obstáculos poseen trayectorias horizontales con comportamiento de rebote en los límites del entorno.

Sesion6_Mini:Videojuego bidimensional que combina mecánicas de recolección y evasión. El jugador controla una nave espacial que debe recolectar objetos verdes para acumular puntos mientras evita colisionar con obstáculos rojos en movimiento que reducen sus vidas.
